# Documentacion del backend CRIS Antártico
Este documento busca ser la puerta de entrada a la documentacion del sistema CRIS Antártico, 
dando una vista general de lo que hay en cada documento, para que el desarrollador o mantenedor 
tenga una idea de lo que se puede encontrar.

Dentro de esta carpeta se encuentran X archivos `.md` en donde se resumen los detalles mas importantes 
para levantar, desarrollar y mantener la aplicacion.

## 02_configuracion_instalacion
En este archivo se indican los detalles necesarios para la configuracion e instalacion del backend, ademas 
de algunos comandos utiles para cargar configuraciones.

## 03_entidades_formularios
En este archivo se indican los archivos para configurar las diferentes entidades que componen el sistema 
CRIS, editando los formularios asociados y los datos que se solicita a cada una.
